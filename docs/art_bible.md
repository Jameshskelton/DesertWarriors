# Desert Warriors Art Bible

## Visual Direction

- 8-bit-inspired pixel presentation with readable silhouettes over literal NES restrictions
- Internal target: `320x180`
- Terrain tiles: `16x16`
- Map units: `24x24`
- Battle sprites: `64x64`

## Palette

- Forest greens for the Greenwood map
- Muted steel blue for George's force
- Rust red for enemy units
- Warm parchment and brass for UI accents

## UI Rules

- High-contrast panels with thick borders
- Large labels over dense iconography
- Forecast windows must surface terrain bonuses clearly

## Animation Rules

- Keep attack cutaways short and readable
- Distinct timing for sword slash, lance thrust, axe cleave, fire cast, and heal cast
- Use screen flash and HP bar movement for feedback before final sprite polish exists
